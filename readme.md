# Simulation

Консольное приложение "simulation" демонстрирует пошаговую симуляцию места, населённого хищниками и травоядными. Задача хищника съесть травоядного.
Задача травоядного - съесть траву и не нарваться на хищника.
     

## Инструкция

1. Предварительно установите в терминале кодировку UTF-8 для корректного отображения объектов.
2. Запустите приложение.
3. Указвть размерность поля симуляции
4. Каждые 5 интераций преостанавливаем симуляцию и уточняем дальнейшее действие:

   "E" - завершить симуляцию

   "Y" - продолжить симуляцию

   "C" - перемещаем конкретный объект на симуляции. Предварительно указав координаты объекта

6. Отслеживайте прогресс симуляции на отображаемом интерфейсе.
